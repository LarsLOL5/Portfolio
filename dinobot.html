Dinobot:
The fast bite

Gameplay Video

Our last project of the first year. This time we were with 5 people, 2 game developers and 3 game artists. For this project, we had 8 weeks. For the first 4 weeks, we needed to build an alpha version with a lot of requirements from school. In the last 4 weeks though, we were free to build whatever we liked.

What challenges did I face?

The biggest problem faced was with the making of the upgrade system. Those scripts were a real challenge. I spend a long time on them, maybe a bit too long.

What did I make?

I made a lot of scripts for this project. But the ones that I am proud of are my upgrade system scripts and my central pickup system scripts.
