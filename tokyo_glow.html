Tokyo Glow: Neon Night

This was our most recent group project made with artists. And it was the biggest group so far, with 7 people total. Before this project we had 2 weeks practice on how to build the car and tweek it. So I came prepaired going into this project. In this 4 week project we make an night-themed street circuit that is based on the real streets of Tokyo. So thats why we chose the name.


What problems did I face?

The communication with some artists wasn't too great. We both had different ideas for the game so it was hard to go on one line.


What did I make?

Mine main task for this project was the player. I needed to "build" the car and let it drive smoothly. It also needed to have a boost.
